## Build AI systems, not AI sessions.

Open-source tools for Claude Code - from session memory to multi-perspective analysis. Each tier works independently. No hard dependencies. No lock-in.

### The Ecosystem

```
You're here          You want this            Install this
-----------          -------------            ------------
Raw Claude Code  ->  Session memory       ->  Starter System
                 ->  Workflow skills      ->  + Skills Bundle
                 ->  Deep planning        ->  + UPF
                 ->  Deep analysis        ->  + Quantum Lens
                 ->  AI-powered memory    ->  + Kairn
```

| Tool | What It Does |
|------|-------------|
| [**Starter System**](https://github.com/primeline-ai/claude-code-starter-system) | Session memory, handoffs, context awareness |
| [**Skills Bundle**](https://github.com/primeline-ai/primeline-skills) | 5 workflow skills: debugging, delegation, planning, code review, config architecture |
| [**UPF**](https://github.com/primeline-ai/universal-planning-framework) | Universal Planning Framework - 3-stage deep planning that catches gaps early |
| [**Quantum Lens**](https://github.com/primeline-ai/quantum-lens) | 7 cognitive lenses + solution engineering - analysis that sees what you don't |
| [**Kairn**](https://github.com/kairn-ai/kairn) | Context-aware knowledge engine - knowledge graph with intelligent context routing |

All tools are MIT licensed. All work with Claude Code out of the box.
